# Simpulator

Simpulator is a extensible 3D simulator engine built using OOP principles.

The engine code is in `core/src/main/java/com/simpulator/engine`.
Demo code is in `core/src/main/java/com/simpulator/game`.

## Demo

There is a demo application that showcases the engine's capabilities. The demo is meant for desktops.

Run the demo via Gradle:

```
gradlew.bat lwjgl3:run  # Windows
./gradlew lwjgl3:run    # Linux/MacOS
```
